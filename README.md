# Tutorial
## For cmd
```bash
> git clone https://github.com/kamikaze2325/KamiBotz
> cd KamiBotz
> npm install
> node .
```
## For Termux
```bash
> termux-setup-storage
> apt update && apt upgrade
> pkg install nodejs
> pkg install git
> pkg install bash
> git clone https://github.com/kamikaze2325/KamiBotz
> cd KamiBotz
> bash install.sh
> npm install
> npm start/node main
```
